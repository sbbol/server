# Поломка  утеря носителя ключей ЭЦП

Поломка / утеря носителя ключей ЭЦП
Сломался, или потерялся ключ электронной подписи – последовательность действий.
В случае утери / поломки ключа ЭЦП:
1. Связаться с отделением банка по месту обслуживания и сообщить о поломке/утере ключа для его блокировки на стороне Банка.
2. В отделении по месту обслуживания получить новый носитель ключей ЭЦП (услуга  может быть платной, подробности уточняйте в отделении Банка)
3. После получения ключа требуется создать запрос на новый сертификат согласно руководству по генерации ключей ЭЦП. Утоните в Вашем отделении, какой по счету сертификат будете получать, и укажите соответствующий номер при заполнении поля «Идентификатор абонента» (пример: «123456789.х» 123456789 – УНП организации, x – порядковый номер).
4. После предоставления в Банк карточки открытого ключа и заявления о присоединении к УЦ (в соответствии с руководством), ориентировочно на следующий рабочий день будет зарегистрирован новый сертификат.
5. Настроить новый сертификат на стороне Клиента для подсистемы Банк-клиент или АС «SavEx».
В подсистеме Интернет-клиент дополнительная настройка не требуется.

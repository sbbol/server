Ошибка – Невозможно подписать документы
Программное обеспечение для подписания документов ClientSign не запущено, либо не установлено. Запустите ClientSign, либо установите его, предварительно скачав архив сертификатов для Вашей учетной записи (Certificate) и файл установки программы (ClientSign).
1. Установить программное обеспечение для подписания документов ClientSign.
Памятка по настройке подписи в «СберБанк Бизнес Онлайн»
799 kb
2. Возможно, плагин ClientSign не был добавлен в автозагрузку при установке. Необходимо запустить из меню пуск или по ярлыку на рабочем столе.
3. Возможно, при установке плагина ClientSign был заблокирован файл hosts. В нем должна присутствовать запись «127.0.0.1  sign.client.bps-sberbank.by». Блокировка hosts могла быть вызвана антивирусом.
